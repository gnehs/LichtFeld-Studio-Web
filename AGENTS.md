## 開發規範

1. 套件管理：使用 `pnpm`（workspace/monorepo）。
2. 主要指令：
   - `pnpm -r build`
   - `pnpm test`
3. 前端：React + Tailwind。
4. 後端：Node.js + TypeScript。
5. Docker 部署：以單一 `./data:/data` volume 為預設。
